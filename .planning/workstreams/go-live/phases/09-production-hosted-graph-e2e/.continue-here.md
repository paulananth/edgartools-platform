## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Trusting background-task exit code through a `tee` pipeline | A background `bash ... | tee file.log` run reported exit 0 even though the underlying script failed, because `tee` (not the script) determined the reported exit code. | advisory | Always grep the actual output content (`FAILED`/`SUCCEEDED`/stage names) rather than trusting a piped exit code. |
| Heredoc `git commit -m "$(cat <<'EOF' ...)"` / `gh pr create --body "$(cat <<'EOF' ...)"` breaking the Bash tool | Multi-line heredoc bodies with parentheses/special characters repeatedly caused `unexpected EOF` / `syntax error: unexpected end of file` in this session (both for `git commit` and `gh pr create`). | advisory | Use `git commit -m "title" -m "body"` (plain strings, no heredoc) or write the body to a temp file and pass `--body-file <path>` to `gh pr create`. |
