|----------|-------------|
| Operator CLI -> MDM source config | `MDM_SILVER_DUCKDB` enters the process from environment/config and can name a local path or URI. |
| Silver DuckDB -> MDM SQL tables | Read-only silver facts are converted into mutable MDM rows. |
| Bronze artifact registry -> parser | Stored primary XML bytes are decoded and parsed into silver ownership rows. |
| Test fixture -> production implementation | Tests define security behavior before production code is repaired. |
