## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Operator browser -> Streamlit app | Local operator actions trigger cached status/smoke reads and navigation. |
| Streamlit app -> read-only helpers | UI crosses into MDM and optional Neo4j helper modules. |
| Environment -> UI diagnostics | Secret-bearing configuration affects status display and must not leak. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-07 | Tampering | `examples/mdm_graph_dashboard/streamlit_app.py` | mitigate | UI exposes only `Refresh data`, imports read-only helpers, and architecture tests block mutation labels/buttons. |
| T-08-08 | Information Disclosure | Streamlit status/error rendering | mitigate | Render exact UI-SPEC fixed copy and helper-provided safe statuses; never render raw env values, DSNs, URIs, passwords, secret JSON, or raw exceptions. |
| T-08-09 | Elevation of Privilege | Dashboard scope/import boundaries | mitigate | Architecture tests block MDM pipeline, graph sync, migrations, generated deployment JSON, Terraform, Step Functions, rollout, Snowflake/dbt, and mutation path references. |
| T-08-10 | Denial of Service | Dashboard smoke output | mitigate | Cache read-only result sets for 60 seconds and keep smoke output hard-bounded to tiny helper result sets; do not expose Phase 9/10 filters. |
| T-08-SC | Tampering | npm/pip/cargo installs | accept | No package-manager install task is planned; existing `uv` extras are used. |
