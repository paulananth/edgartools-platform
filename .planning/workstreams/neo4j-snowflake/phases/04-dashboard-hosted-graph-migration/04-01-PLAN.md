<interfaces>
The helper should expose a primary hosted graph function, for example `get_snowflake_graph_metrics(...)`, returning a dataclass with `as_dict()`.

Required dictionary shape:
- `available`
- `state`
- `message`
- `last_refreshed`
- `target`
- `snowflake_graph_nodes`
- `snowflake_graph_edges`
- `entity_comparison`
- `relationship_comparison`
- `diagnostics.missing_graph_nodes`
- `diagnostics.extra_graph_nodes`
- `diagnostics.missing_graph_edges`
- `diagnostics.extra_graph_edges`
- `diagnostics.missing_graph_edge_endpoints`
- `native_app.status`
- `native_app.failing_checks`

Each comparison row must include counts, mismatch deltas, and a status. Endpoint diagnostic rows must carry enough fields for the UI to render `source -> target`.
</interfaces>
