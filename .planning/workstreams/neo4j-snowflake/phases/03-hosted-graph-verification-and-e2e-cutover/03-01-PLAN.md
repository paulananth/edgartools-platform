<interfaces>
Existing `verify-graph`:
- Reads `SnowflakeConnectionSettings.from_env()`.
- Defaults to `DEFAULT_TARGET_SCHEMA = "NEO4J_GRAPH_MIGRATION"`.
- Counts `MDM_GRAPH_NODES` and `MDM_GRAPH_EDGES`.

Required Phase 3 verifier:
- Reads active MDM entities from `MDM_ENTITY` joined to active `MDM_ENTITY_TYPE_DEFINITION`.
- Reads active MDM relationships from `MDM_RELATIONSHIP_INSTANCE` joined to active `MDM_RELATIONSHIP_TYPE`.
- Compares those expected rows against `MDM_GRAPH_NODES` and `MDM_GRAPH_EDGES`.
- Emits secret-safe JSON and returns nonzero for any mismatch.
</interfaces>
</context>
