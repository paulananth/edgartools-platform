## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention |
|---|---|---|---|
| Heredoc commit/PR messages | `git commit -m "$(cat <<'EOF' ... EOF)"` and `gh pr create --body "$(cat <<'EOF' ...)"` both throw `unexpected EOF while looking for matching` in this shell. | advisory | Write message to `/tmp/*.txt`/`*.md` via Write tool, then `git commit -F <file>` / `gh pr create --body-file <file>`. |
| `gh pr merge --delete-branch` deletes local branch + unpushed commits | If you make local WIP commits on a PR branch, push them before merging, or the local branch+commits get deleted (recoverable via `git reflog` + `git fsck --unreachable`). | advisory | Push WIP commits before merging, or omit `--delete-branch`. |
